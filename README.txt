Library Index module is the common Api towards api.kirjastot.fi to get
information about libraries.
Library Open Hours retrieves open hour information from api.kirjastot.fi.
Library Information retrieves basic information from api.kirjastot.fi
 - Library name
 - Open hours, if open hours module enabled and selected to show
 - Library picture
 - Library telephone numbers
 - Visit address
 - Postal address

Module uses field cache with settable cache period.
Consortiun library lista can be used to select library.
